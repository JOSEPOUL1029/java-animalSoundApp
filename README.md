# Zoo Animals - Java OOP Assignment

## Overview
This project is a simple Java application that demonstrates Object-Oriented Programming (OOP) concepts by simulating a zoo with different types of animals.

The application includes:
- Dog
- Cat
- Bird

All animals can make sounds, while only domesticated animals can participate in play activities with visitors.

## Objectives
- Implement **Abstraction** using an abstract `Animal` class.
- Use an **Interface** to represent play activities for domesticated animals.
- Demonstrate **Inheritance** by creating different animal classes.
- Apply **Runtime Polymorphism** by calling overridden methods through the parent class reference.
- Use **Arrays** and **Loops** to store and display information about multiple animals.

## Features
- Display the sound made by each animal.
- Show play activities for domesticated animals.
- Demonstrate Java OOP principles in a simple and structured program.

## Technologies Used
- Java
- Object-Oriented Programming (OOP)

## Project Structure
```
src/

```

## How to Run
1. Clone this repository.
2. Open the project in your preferred Java IDE (IntelliJ IDEA, Eclipse, or VS Code).
3. Compile and run `Main.java`.
4. View the animal sounds and play activities in the console.
